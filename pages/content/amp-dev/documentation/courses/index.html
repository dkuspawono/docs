About the courses

Beginning web developers will learn to create modern websites with AMP. Experienced developers will discover how to use AMP to create beautiful, interactive sites while writing less JavaScript. During the course, students build a lovely site for Chico's Cheese Bicycles.

Requirements

  • Internet connection
  • Chrome browser, ideally, to use the AMP Validator extension
  • We use the online editor Glitch. No IDE or local web server needed!

Prerequisites

  • basic knowledge of HTML and CSS
  • for the advanced course, JavaScript expressions and simple JSON
